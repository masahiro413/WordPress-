# WordPress 病院向けブログサイト

病院・クリニック向けの WordPress カスタムテーマです。  
診療科・医師紹介・お知らせブログを管理でき、患者向けに分かりやすい情報を提供します。

---

## 主な機能

| 機能 | 説明 |
|---|---|
| **カスタム投稿タイプ: 診療科** | 診療科ごとのページを作成。アイコン絵文字・電話・診療時間を設定可能 |
| **カスタム投稿タイプ: 医師** | 医師ごとのプロフィール（役職・専門・経歴・担当曜日）を管理 |
| **ブログ / お知らせ** | WordPress 標準の「投稿」を利用した記事管理 |
| **テーマカスタマイザー** | 病院名・電話番号・住所・診療時間・お知らせバーをノーコードで設定 |
| **レスポンシブデザイン** | PC・タブレット・スマートフォンすべてに対応 |
| **日本語対応** | Noto Sans JP / Noto Serif JP フォントを使用 |
| **アクセシビリティ** | スキップリンク・ARIA ラベル・セマンティック HTML |

## ディレクトリ構成

```
├── archive.php            # アーカイブ（カテゴリ/医師/診療科一覧）

├── sidebar.php            # サイドバー

├── inc/
│   ├── meta-departments.php   # 診療科メタボックス

    └── main.js            # モバイルメニュー・スムーズスクロール
```

## インストール手順

1. `hospital-theme/` フォルダを WordPress の `wp-content/themes/` にコピーします。
2. 管理画面「外観 → テーマ」から **病院ブログ** を有効化します。
3. 「外観 → カスタマイズ → 病院情報」で病院名・電話番号・住所などを設定します。
4. 「外観 → メニュー」でナビゲーションメニューを設定します。
5. 「診療科」「医師」の投稿タイプにデータを追加します。
6. 「固定ページ」で「ホームページ」を作成し、「設定 → 表示設定」でフロントページに指定します。
